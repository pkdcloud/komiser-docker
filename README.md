# Komiser Docker Container

This repository provides a minimal Docker setup to run Komiser from a scratch image.

## Requirements

- Docker
- Docker Compose V2
- make

## Usage

1. Run the container:

   ```sh
   make run
   ```

2. Stop the container:

   ```sh
   make run
   ```

3. Clean:

   ```sh
   make run
   ```

## Access Service

You can Access the service at http://localhost:3000
